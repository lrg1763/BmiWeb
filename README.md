# Калькулятор ИМТ

Одностраничный сайт для расчёта индекса массы тела (ИМТ). Адаптивная вёрстка, ввод только с клавиатуры.

## Возможности

- Ввод веса (кг) и роста (см) с клавиатуры (без стрелок и колёсика мыши)
- Расчёт ИМТ по формуле: вес ÷ рост² (м)
- Интерпретация по категориям: дефицит, норма, предожирение, ожирение 1–3 степени
- Шкала ИМТ для наглядности
- Цветовая схема: чёрный, белый, жёлтый
- Адаптивность под мобильные и десктоп

## Технологии

- HTML5
- CSS3 (Flexbox, медиазапросы)
- JavaScript (ES5)

## Запуск локально

Откройте файл `index.html` в браузере или запустите локальный сервер в папке проекта:

```bash
npx serve .
```

или

```bash
python3 -m http.server 8000
```

После этого откройте в браузере: `http://localhost:8000` (или адрес, который выведет сервер).

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub.
2. Запушьте код в ветку `main`.
3. В репозитории: **Settings** → **Pages** → **Source**: ветка **main**, папка **/ (root)** → **Save**.
4. Сайт будет доступен по адресу: `https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПО/`.
