# Калькулятор ИМТ

Веб-приложение для расчёта индекса массы тела (ИМТ). Минималистичный дизайн, адаптивная вёрстка, без внешних зависимостей.

## О проекте

Калькулятор позволяет быстро оценить соответствие веса и роста по формуле ВОЗ. Результат отображается с интерпретацией и цветовой шкалой. Ввод данных — только с клавиатуры (стрелки и колёсико мыши отключены).

## Возможности

- **Расчёт ИМТ** по формуле: `ИМТ = вес (кг) / рост² (м)`
- **7 категорий интерпретации:**
  - 16 и менее — выраженный дефицит массы тела
  - 16–18,5 — недостаточная (дефицит) масса тела
  - 18,5–25 — норма
  - 25–30 — избыточная масса тела (предожирение)
  - 30–35 — ожирение первой степени
  - 35–40 — ожирение второй степени
  - 40 и более — ожирение третьей степени (морбидное)
- **Визуальная шкала** с цветовой индикацией диапазонов
- **Валидация** ввода (вес 20–300 кг, рост 100–250 см)
- **Адаптивность** — корректное отображение на мобильных и десктопах
- **PWA** — поддержка favicon и web manifest

## Технологии

| Технология | Назначение |
|------------|------------|
| HTML5 | Разметка, семантика |
